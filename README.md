# Sistem Pakar Rekomendasi Media Tanam

Website sistem pakar berbasis rule IF-THEN untuk merekomendasikan media tanam yang sesuai berdasarkan jenis tanaman, lokasi penempatan, dan intensitas cahaya.

## Tentang Project

Project ini dibuat sebagai tugas kelompok 5 mata kuliah Sistem Berbasis Pengetahuan kelas 3KA06 Universitas Gunadarma.

Sistem menggunakan pendekatan rule-based expert system dengan inference engine menggunakan JavaScript.

## Fitur

* Pemilihan tanaman melalui katalog interaktif
* Pemilihan lokasi penempatan
* Pemilihan intensitas cahaya
* Sistem inferensi berbasis IF-THEN
* Hasil rekomendasi media tanam
* Penjelasan media tanam terpilih

## Input Sistem

Sistem menerima 3 input utama:

1. Jenis Tanaman
2. Lokasi Penempatan
3. Intensitas Cahaya

## Output Sistem

Sistem memberikan:

* karakteristik tanaman
* rekomendasi media tanam dan penjelasannya

## Teknologi yang Digunakan

* HTML
* CSS
* JavaScript
* GitHub Pages

## Basis Pengetahuan

Sistem menggunakan:

* 12 jenis tanaman
* 6 media tanam
* 72 rules IF-THEN

## Cara Menjalankan

1. Buka lini https://aryudaa.github.io/expert-system-media-tanam/
2. Pilih tanaman
3. Pilih kondisi lingkungan
4. Klik tombol Cari
